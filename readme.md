# Coconuts.finance

## Overview

Frontend repo of [Coconuts.finance](https://docs.coconuts.finance/). 

## Env

Required env vars:

```
REACT_APP_NETWORK_ID = 137 // Network ID of BSC
```

## To Run

```
yarn
yarn start
```

The app will be running at http://localhost:3000/
