# RoboVault

## Overview

Frontend repo of [RoboVault](https://docs.robo-vault.com/). 

## Env

Required env vars:

```
REACT_APP_NETWORK_ID = 250 // Network ID of BSC
```

## To Run

```
yarn
yarn start
```

The app will be running at http://localhost:3000/
